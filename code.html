									//Initialize empty array

REWIND

Loops is an app that loops through a string entered by a user abd reverses the position of each character. The code contains three functions: getValues(), reverseString(), and displayString().

Function: getValues()

  1. This function gets the value of the string entered in the from by a user.
  2. The value is assigned to a variable called userString.
  3. The function reverseString() is called and assigned to the variable revString which will contain the reversed characters after they are processed by the calling function.
  4. Then, displayString() function is called which will display the reversed characters on the screen.
  5. The first statement in this function block adds an "invisible" class to the div with the id "alert" which will be removed to reveal a status message after the string has been reversed.

Function: reverseString()

  1. The purpose of this function is to reverse the string. It does that by first initializing an empty array called revString.
  2. Next, a For loop is used to loop through the userString.
  3. Because we are reversing the string, we have to start the index at the last character of the string. Therefore, the index is set to the length of userSring minus one (index = userString - 1).
  4. For the next parameter in the For loop, index is set to >= o (index >= 0). This is because we are going backwards from the last index in the string to the first.
  5. Likewise, the index is then decremented by 1 (index--) in order to get to evaluate and reverse the next character in the loop.
  6. Each time through the loop, the individual reversed character is added to the variable, revString.
  7. When the loop is finished processing and all reversed characters are added to revString, revSring is returned to the calling function, reverseString() which assigns it to its local variable, also called, revString.

Function: displayString()

  1. The purpose of this function is to display the reversed string on the HTML page. This is accomplished by assigning the reversed string to the id "msg" via the innerHTML property of the getElementById method of the DOM.
  2. Then, the "invisible" class is removed from the "div" tag with the "alert" id to display the status message on the page.
